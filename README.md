# Learning Playwright Fundamentals 2X

This repository contains a Playwright-based test project for learning browser automation fundamentals.

## Project Overview

The project includes:
- Playwright configuration in playwright.config.ts
- Example tests in the tests/ directory
- Test report output in playwright-report/ and test-results/

## Prerequisites

Make sure you have Node.js installed on your machine.

## Setup

Install dependencies:

```bash
npm install
```

Install the Playwright browsers:

```bash
npx playwright install
```

## Running Tests

Run the test suite:

```bash
npx playwright test
```

Generate and open the HTML report:

```bash
npx playwright show-report
```

## Notes

This project was created as part of a Playwright fundamentals learning exercise and is intended for practice and experimentation.
